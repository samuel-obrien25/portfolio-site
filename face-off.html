Project Notes

Face Off is a progressive web app for quick access to NHL schedules and stats. It is currently a work in progress, being developed with React. The latest build allows users to see the schedules and past game scores for each NHL team.

Future builds will feature player stats, team sorting, and more game data.

Check out the latest build. Please note: It may take up to 30 seconds for app to load due to limitations on Heroku's free tier.
